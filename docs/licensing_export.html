U Upgrade Trigger Summary
Free → Plus (₹99/mo)
  • Hitting the 10-reply/day cap regularly
  • Need history beyond 7 days
  • Want cross-device sync between phone and web
  • Need to export conversations as PDF/CSV
  • Want premium Azure Neural TTS voices
Plus → Pro (₹149/mo)
  • Want emotion trend charts and mood graphs
  • Need unlimited history (beyond 90 days)
  • Want weekly summaries and monthly growth letters
  • Interested in long-term growth arc tracking
  • Want weekly insight digest notifications
Pro → Family (Custom)
  • Multiple household members sharing one account
  • Need child-safe profiles for minors
  • Parent wants oversight via parental controls
  • Need profile-level privacy isolation on shared device
  • Want family-wide usage dashboard
Pro → EDU (Custom)
  • Institution deploying for students or staff
  • Need admin panel with usage analytics
  • Require LMS integration (Moodle, Canvas)
  • Need bulk user provisioning and SSO
  • Data residency or compliance requirements
Pro / EDU → Enterprise (Custom)
  • Corporate / HR / mental-health platform deployment
  • Need REST API access for custom integrations
  • Require child-safe mode for enterprise wellness programs
  • Need team/department management with sub-admins
  • Bespoke HR system connectors (Workday, SAP)
  • Data residency, SOC 2 / ISO 27001 compliance
  • Contractual SLA and dedicated account manager
  • Custom AI model tuning for org-specific tone
